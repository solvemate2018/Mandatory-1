Terminal documentation:


How to list all files in a folder?

The command for that is "ls".


How to change the folder you are located in?

The command for that is "cd [path]" where "path" can be relative or absolute.


How to move files?

The command for that is "mv [file name] [path]" where "file name" is the name of the file you want to move, and [path] is where you want to move the file.


How to delete files?

The command for that is "rm [file name]"


How to make a new folder?

The command for that is "mkdir [name]" where "name" is the name of the folder.


How to run a Node application?

The command for that is "node [js file]"


How to stop a running Node server?

In order to stop the server you need to press both CTRL + C
