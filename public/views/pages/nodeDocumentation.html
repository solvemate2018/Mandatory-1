Node documentation


What is Node.js?

Node is a JavaScript runtime. Not a framework/library! Not a JavaScript compiler!


What is Express?

Express is a web framework for Node.js. It's fast, minimalist and unopinionated!


How to beggin programming with Node.js?
  1. Create a new folder for your Node application.
  2. Create a file named "package.json" which will include our dependancies.
  3. Write the dependancies which you want to use in your application.
  4. Open a terminal and write "npm install"
  5. Wait until the NodePackageManager(npm) installs all packages
  6. Create a file named "app.js" and start exploring the Node world
